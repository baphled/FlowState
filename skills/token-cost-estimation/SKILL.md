# Token-Cost-Estimation Skill

Estimate token costs upfront before starting work. Generate a breakdown by phase (investigation, implementation, verification) to understand your token budget and identify optimization opportunities.

Invoke at session start with goal, complexity tier, and estimated duration. The breakdown format shows tokens per phase and total budget. Use this to make conscious trade-offs: skip unnecessary investigation, parallelize independent work, or reduce scope if budget is tight.

Integration with token-efficiency and scope-management ensures you stay within budget while delivering quality work. Track actual vs estimated costs to improve future estimates.
